# Anchor list

A list of anchors. Items that contain links to the current page are
marked as active. Useful for navigation menus.

<Preview withToolbar>
  <Story name="AnchorList">
    <AnchorList items={[
      { href: '/', text: 'Inactive 1' },
      { href: '#main-content', text: 'Active' },
      { href: '/#main-content', text: 'Inactive 2' }
    ]} />
  </Story>
</Preview>

<Props of={ AnchorList } />


## How it works

An item whose link is for the current page will have a an `active` BEM
class modifier put on it allowing you to style it in your CSS / SASS
code.
