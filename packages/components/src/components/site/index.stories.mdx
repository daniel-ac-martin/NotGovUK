# Site

A component for defining an entire web site.

<Preview withToolbar>
  <Story name="Site">
    <Site
      phase="alpha"
      title="My service"
      titleHref="#my-service"
    >
      My site.
    </Site>
  </Story>
</Preview>

<Props of={Site} />


## How it works

In effect this defines multiple pages and automatically does some basic work to
wire them together.

<Preview>
  <Story name="Standard">
    <Site
      feedback={{
        content: (
          <h1>Feedback</h1>
        )
      }}
      phase="alpha"
      title="My service"
      routes={[
        {
          href: "/one",

        {

      <StartButton href="/one" />

  </Story>
</Preview>
